# Kettle 装配说明

Kettle 的重写版本把壶身、盖、把手和壶嘴分别看成独立的局部装配模板。每个模板先在自己的局部坐标系内完成轴向堆叠、端点连接或弧段拼接，最后再由 concept 级 `position` 和 `rotation` 放到整壶坐标中。

壶身模板围绕中心竖轴构建。球面壶身使用截断椭球片表达上、下壳体轮廓，底部足圈贴到下壳体投影出的连接截面。多层壶身则从底部外壳开始向上堆叠，每一层的顶部半径自然成为下一层的底部半径，保证轮廓连续。

盖子模板以底面为局部原点。下部开口环先落在原点，上部封盖贴到开口环顶部，旋钮段再逐层接在封盖顶面上，整体形成一个从开口到顶部把手的竖向链。

把手模板按实际结构分成三种关系：单段弧形或直杆直接由一个 primitive 表达；U 形把手由两根立柱和顶部半环组成；三折把手先确定上下两根安装杆的外端，再用一根后侧桥接杆连接这两个端点。

壶嘴模板分为直线分段和弧线分段。直线壶嘴把每个管段看成沿自身方向延伸的短管，按顺序推进管段中心。弧线壶嘴先放置上弧，再把下弧的起点锚到上弧终点，使两段喷嘴在中心线上连续。
