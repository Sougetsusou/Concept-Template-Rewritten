# Doorhandle 装配思路

Doorhandle 的三个模板都把门面看作局部坐标中的安装平面，所有部件从这个平面沿法线方向向外生长。固定底座先贴住门面，后续部件不再手写中心点坐标，而是把自己的贴合面移动到前一部件的外侧接触面。

长方体型把手由贴门固定板、第一段握持件和第二段握持件组成。固定板的背面位于安装平面，后两段依次以前一段的外侧面为安装面，并允许在门面平面内有偏移。

旋钮型把手由三个同轴圆柱组成。每个圆柱的轴线沿门面法线方向，底座贴门，颈部接在底座外端，旋钮主体接在颈部外端。

T 形把手先用圆柱 stem 从门面伸出，再把外侧横杆的背面贴到 stem 端面；横杆可以在门面平面内偏移，用来表达 T 形头部不完全居中的情况。

整件把手的最终位置和姿态作为 concept 级变换处理，保留先平移再旋转的外部摆放约定。
