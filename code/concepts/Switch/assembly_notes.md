# Switch 装配说明

Switch 的 rewritten 模板把整体看作“面板平面上的底座、开关件和插头触点”。所有 concept 的最终姿态仍由外层 `position` / `rotation` 控制，内部装配先在局部面板坐标系中完成。

底座类只表达面板背后的实体结构：标准底座是一块前表面对齐面板平面的主体，可选背部块继续沿深度方向接在后方；框形底座使用一个矩形环表示开孔框；平行底座用两条左右块共享同一个前参考平面。

旋钮和圆形开关都是沿面板法向伸出的圆柱。单旋钮把圆柱底面贴到面板面上；多圆形开关保留每个按钮的中心布局位置，并允许整组按钮有统一的倾角。

翻板开关是沿 X 方向排布的重复长方体。FlipX 和 FlipY 的区别不是新的物体类型，而是翻板绕自身不同轴倾斜，因此保留为两个语义明确的模板。

Lever 开关由圆柱底座和细长杆组成。底座先贴在面板上，杆的底面再挂到对应底座的顶面安装点；杆的倾斜由自身旋转表达，不再对已经生成的 mesh 顶点做二次变换。

插头类是面板背后的触点。Cuboidal 和 Cylindrical 版本分别使用长方体或圆柱触点网格；三插脚版本保留一个中心脚和左右两个侧脚，各脚先用前表面 anchor 放到局部布局点，再整体绕插头中心旋转。
