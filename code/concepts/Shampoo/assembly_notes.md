# Shampoo 装配设计说明

Shampoo 的瓶身主要分为两类：一类是沿竖直轴逐段堆叠的圆柱或锥台轮廓，另一类是单个长方体或软管式压扁轮廓。分段圆柱瓶身把每一段看作一个完整截面，后一段的底部贴到前一段顶部，截面比例在整条轮廓中保持一致。

软管式主体不是多部件装配，而是一个上端较圆、下端压扁的渐缩体。它保留一个很小但非零的底部厚度，用来表达封尾的窄边并避免退化成零面积截面。

喷嘴由竖向底座和侧向喷嘴段组成。底座按圆柱段从下往上生长；每个喷嘴段用自身的后端面贴到上一段前端或底座外缘，再按给定俯仰角确定前端位置，形成连续的喷嘴链。整条喷嘴链最后绕瓶身竖直轴调整朝向。

圆柱盖由一个带内腔的环形壳体和一个顶部填充段组成。壳体定义总高度和内外轮廓，填充段把上方封住并留下下方内凹空间。普通盖则只是单段圆柱体。
