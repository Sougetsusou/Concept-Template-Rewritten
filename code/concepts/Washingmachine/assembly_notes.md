# Washingmachine Assembly Notes

## 设计思路

滚筒式机身被表达为“实心支撑段 + 带圆孔的外框段”。前开式机身沿深度方向拼接，顶部开口式机身沿高度方向拼接；开口本身属于方外圆孔 primitive 的局部形状，而不是额外圆柱部件。

矩形机身被表达为底板托住上方矩形中空壳体。底板和壳体使用面到面的 anchor 贴合，壁厚只负责定义中空壳体的内轮廓。

滚筒门使用同一个门板局部 frame 放置外环、中间环和中心圆盘。这个 frame 的安装原点位于门外接区域的左后下参考角附近，因此门的圆心在局部 frame 内向右、向前偏移；这表示门板在安装坐标系里的占位，不是额外补偿。中间环和中心圆盘共享偏心量，用来表达门面中偏置的透明或装饰区域。

矩形门板以安装面作为局部原点，门板实体从该面向外伸出。

控制面板先建立一个楔形斜面板，再把每个按钮的背面贴到斜面上的安装点。按钮厚度只影响按钮自身尺寸，不再混进斜面安装点的计算。
