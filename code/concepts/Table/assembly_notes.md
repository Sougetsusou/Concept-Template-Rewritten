# Table 重写装配说明

Table 的 rewritten 版本把整类物体拆成几组自然装配 recipe：桌面、桌腿、层板、门抽屉、柜体和隔板。桌面类是单块板或圆柱面；层板类是在竖向参考线上重复放置板件或圆形层面；背板和隔板类以安装面为参考，把板件贴到对应的顶面、侧面或后面位置。

桌腿类保留“中心布局间距”作为模板语义：参数描述腿中心在平面上的阵列位置，primitive 负责用 top anchor 把腿的安装高度对齐到桌面参考平面。带横撑的腿组先建立主腿，再从主腿自身的局部安装点取得横撑端点；横杆使用两点轴向对齐，把杆件的局部轴直接连接到两个安装点。

斜拉腿组由四条主腿和两根对角连接杆组成。连接杆不再用固定原点和手算 yaw 表达，而是从两条对角腿的 top anchor 派生杆件中心线端点，再用两点轴向对齐完成装配。

星形腿由中心支柱和一组径向脚组成。中心支柱定义竖向支撑，径向脚按等角度径向阵列分布，并带有局部倾角；这是一类“中心支撑 + 径向重复脚”的装配关系。

吧台式桌腿由竖向支撑和底部横脚组成。竖向支撑先确定主承重方向，横脚再把自身顶面挂到支撑底面，表达“脚托接住支撑”的接触关系。

抽屉和门都是面板式子装配。抽屉由左右侧板、前后板、底板、前板和可选把手组成；门由旋转门板和门板表面的把手组成。把手属于门或抽屉表面的附件，不单独提升成全局 relation。

柜体是 Table 中最复杂的子装配。rewritten 版本把它作为“柜体壳 + 层板分隔 + 内嵌抽屉/门”的单个模板处理：先生成外壳侧板、上下板和后板，再根据层板切出的空间填入抽屉或门。当前实现保留柜体模板的整体语义，不把柜体拆成多个独立 category class。
