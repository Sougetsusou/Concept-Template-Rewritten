# Dispenser rewritten 装配说明

本文记录 rewritten Dispenser 的装配设计思路。它面向后续审阅和继续重写，不作为代码逐行注释，也不绑定具体 legacy 字段名。

## 总体设计

Dispenser 的几何可以分成两类：

- 容器主体：由一个整体盒体，或多段竖向圆柱/圆台组成。
- 出液头部：由竖直支撑、倾斜头部、出液口和手柄组成。

重写时优先把部件关系表达为“面贴面”或“端点接端点”的装配语义。只有当某个偏移定义了部件自身的形状姿态时，才保留为当前模板内部的 shape rule。

## 多段圆形主体

设计意图：

多段圆形主体是一条沿竖直方向生长的截面轮廓。底部段定义局部基准，后续段逐层贴到上一段顶部，形成连续的瓶身或罐身。

装配思路：

- 从底部基准段开始构建。
- 每个上方段的底面贴到上一段顶面。
- 半径变化属于每一段的截面形状定义。
- 层间连续性由相邻段的上下截面半径表达，不需要额外 relation。

实现备注：

- 使用 `Cylinder`。
- 上下段通过 `bottom_center` 和 `top_center` 贴合。

## 盒状主体

设计意图：

盒状主体是一个整体实体，不包含内部装配关系。它只需要把语义尺寸映射到几何 primitive 的长、高、深。

装配思路：

- 用一个长方体表达完整主体。
- 局部中心作为主体中心。
- 没有子部件需要额外 anchor 拼接。

实现备注：

- 使用 `Cuboid`。
- 尺寸语义保持为长、高、深，再映射到 primitive 的 length、height、width。

## 按压式出液头

设计意图：

按压式出液头由竖直座体和向外伸出的出液通道组成。座体提供高度和半径基准，出液通道从最上层座体的外侧伸出。如果出液通道有第二段，它接在第一段末端，形成分段通道。

装配思路：

- 先建立竖直座体。
- 座体每一层都贴在上一层顶部。
- 第一段出液通道从座体顶部外侧出发。
- 第二段出液通道如果存在，接在第一段末端。
- 每段通道可以有自己的倾角；这些倾角描述该段自身方向。

实现备注：

- 座体使用 `Cylinder` 的 `bottom_center` / `top_center` 堆叠。
- 出液通道使用 `Cuboid`。
- 第一段通道以 `back_center` 贴到座体外侧安装点。
- 第二段通道以 `back_center` 贴到第一段的 `front_center`。

## 扳机式喷头

设计意图：

扳机式喷头是一个带父子 frame 的结构：底座和中段形成竖直支撑；倾斜头部是喷口和手柄的父部件；喷口贴在头部前端，手柄挂在头部下方。

装配思路：

- 底座定义喷头的局部地面基准。
- 中段贴在底座上方，形成竖直支撑。
- 倾斜头部不是简单地贴在中段顶面；它有自己的前向偏移和倾角规则。
- 喷口的安装点在倾斜头部的前端。
- 手柄的安装点在倾斜头部的下方。
- 喷口和手柄的位置应先在头部局部 frame 中描述，再转换到整个喷头的局部坐标。

实现备注：

- 底座和中段通过 `bottom_center` / `top_center` 竖向贴合。
- 头部保留一个内部 shape rule，用于描述前向偏移和倾角产生的中心位置。
- 喷口使用自身端面 anchor 贴到头部前端安装点。
- 手柄使用自身顶面 anchor 贴到头部下方安装点。
- 头部局部 frame 转换目前是该模板内部能力，不提升为共享 relation。

## 边界

- 本文只描述 rewritten 后的装配设计，不记录 legacy 公式迁移过程。
- 共享 geometry 层只承担 primitive、anchor、transform 和 assembly。
- 当前 category 内部的 shape rule 只有在跨 category 重复出现、语义稳定后，才考虑上升为公共能力。
