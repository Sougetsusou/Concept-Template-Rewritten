# StorageFurniture 装配说明

## 柜体主体

柜体被看作一个开放的矩形壳体：左右侧板定义外部宽度，底板封住下方，背板封住后方。可选顶盖从柜体顶部向上贴合，并允许在三个方向上有少量偏移。

水平层板是插入壳体内部的横向隔板，每块层板用自己的中心高度表达。竖隔板不直接依赖全局坐标，而是属于某个竖向空间；每个空间提供净高、中心高度和横向分布规则，竖隔板按这些空间语义放置。

额外板件被视为柜体局部坐标中的自由板，可表达斜板、补充隔板或附件。它们不参与壳体和层板的自动求解。

## 围合式底座

底座由四块竖板围成矩形框。左右板贴到外包络的左右面，前后板贴到外包络的前后面；角部重叠是这种板式围框的自然结果。额外腿件作为独立支撑块放入同一局部空间。

## 门与把手

门板按给定中心和偏航角放置，旋转发生在门板自身中心。把手使用门板局部坐标中的安装点，然后让把手背面贴到该安装点；这样门板旋转后，把手位置和朝向自动跟随门板。

## 抽屉

每个抽屉是一个浅盒：左右侧壁限定宽度，前后内壁限定深度，底板从下方承托，外前板贴到盒体前侧。把手阵列挂在外前板表面，单把手居中，双把手按左右间距展开。

## 前面板

前面板集合只表达若干独立平板。它不推导间距、不镜像、不附带局部旋转；需要更复杂门板行为时应使用门组件。
