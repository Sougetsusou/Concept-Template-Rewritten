# Microwave 装配说明

Microwave 的 rewritten 版本按功能分成箱体、门板、把手、控制面板和托盘。每个模板只描述自身局部装配，整体位置和姿态作为 concept 级变换统一应用。

箱体由后侧实体板和前侧开口壳体组成。后板先确定完整深度的背面基准，开口壳体再沿深度方向贴到后板前侧，矩形内孔表达微波炉正面的开放空间。

门板有平板和下沉框两种。平板门从背面安装平面向前延伸；下沉门先放置后侧实体层，再把前侧矩形框贴到实体层前面，框内开口可以在门板平面内偏移。

把手都以背面安装平面为基准。直把手是单个长方体；三段式把手用上下两个安装座加前方握持条；弧形把手把握持条替换成一段圆弧，弧段的半角和弧心深度由其覆盖角、半径和安装座间距决定。

控制面板由底板和显式按钮列表组成。底板从安装平面向前延伸，每个按钮根据自己的平面偏移贴到面板前侧，不再使用动态变量名查找按钮参数。

托盘是竖直圆柱，底面作为安装基准向上延伸。
