# Knife 装配说明

Knife 的重写版本把部件分成把手、刀刃、护手和按钮四类。简单把手、护手和按钮都是单个 primitive；它们只表达自身形状，最终姿态由 concept 级 transform 控制。

T 形把手由一个主块和一个下方块组成。主块保持居中，下方块用自己的顶面中心贴到主块底面中心，并允许在安装平面内有少量偏移。

多层把手由底层横块和左右两个上层块组成。底层横块提供承托平面，左右上层块以底面贴到这个平面上，并围绕一条中心线成对分布。

包围式把手是一个开放框架。底梁定义整体下边界，后梁和两条侧梁围成三边框，前侧再用两段短梁绕开中央缺口。每段梁都用自己的边或面锚点贴到外包络的对应边界上。

直刀刃从局部基准平面向上延伸。尖刀刃由带斜向轮廓的根部和退化到尖端的上段组成，上段底面贴到根部顶面。弯刀刃同样以斜向根部为基础，再把四分之一椭圆柱刀尖放到根部末端。

弧形把手和弯刀尖的固定局部旋转属于它们的形状 frame：它们不是额外装配关系，而是把 primitive 的局部成形方向放入 Knife 坐标系。
