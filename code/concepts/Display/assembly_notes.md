# Display Assembly Notes

## 整体思路

Display 的 rewritten 装配把对象拆成三类自然部件：屏幕面板、底座、支撑。所有 class 都保留同一个最终姿态 frame：部件先在 Display 局部坐标中完成拼接，再整体应用 `YXZ` 顺序和 offset-first 的外部变换。

## 屏幕

平面屏幕以主面板作为稳定参考。如果有背部加厚层，加厚层的前表面贴到主面板背表面，并允许在屏幕平面内偏移。梯台屏幕把后壳看成一个从前接触面向后收放的壳体，前侧附加层只在存在时贴到同一个前接触平面。

## 底座

矩形和圆形底座是单个可俯仰的承托板。T 形底座由横杆和向前延伸的主杆组成，两者共享同一个倾斜姿态，主杆用后表面贴到横杆前表面。V 形底座由两个同尺寸翼板组成，它们围绕 Display 局部原点向左右打开。

## 支撑

简单支撑是一排重复的矩形杆。竖直支撑把顶部中心放在安装线上；后向支撑把顶部前边放在安装线上，再按俯仰角向后展开。三段支撑是一条可选链：上部板提供靠近屏幕的安装面，中段从它的后下方继续延伸，底部脚板再接到中段末端并可单独倾斜。T 形支撑则是上横杆加一个从其下方伸出的中段杆。

## 边界

这些装配关系都能由现有 Cuboid/Cylinder anchor 和 primitive transform 表达；当前不需要 Display 专属 relation，也不需要新增 primitive anchor。尺寸比例、可选开关和局部俯仰属于各自模板的 shape/layout 规则，保留在 rewritten concept 内部。
