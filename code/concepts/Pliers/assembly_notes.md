# Pliers 装配说明

Pliers 的 rewritten 版本把钳子拆成轴、把手、夹头和挡板四类局部组件，每个组件都先在自己的局部坐标中完成形状定义，再用 anchor 或局部接触点放到装配关系里。

圆形轴是一个主圆柱，可按需要叠加一个偏移圆柱。矩形轴以一个参考平面为中心：下层矩形块从参考平面向下，上层矩形块从参考平面向上堆叠，并允许每层在水平面内有自己的偏移和 yaw。

直把手由左右两条折线链组成。每侧先放前段，让前段的前面中心落到左右基准点；后段再用前段后侧的边中心作为接续目标。这样把“旋转后端点”从手写三角函数变成了 primitive anchor 的世界坐标。

弧形把手仍保留弧段半径、存在角和弧段端点位移的形状推导。直段和弧段之间用局部内缘接触点贴合，弧段起点接到前直段后缘的单一真实 anchor；中间弧段连接后直段时，后直段继续接到弧段终点 anchor。这里采用的是物体装配语义，而不是 legacy 中把不同侧向投影拼成接触点的写法。

尖形夹头由左右两组楔形块组成。楔形本身用 `Cuboid` 的上下截面和 `top_offset` 表达；左右夹头通过局部接触点固定到各自基准点，再绕开合方向形成对称夹持结构。

弧形夹头和弧形挡板直接使用 partial cylinder / ring primitive 表达。右侧部件的翻转属于镜像朝向的一部分，保留在对应 primitive 的 rotation 中，而不是额外生成临时 mesh。
