# Laptop 装配思路

Laptop 的几何由三个自然部位组成：底座、屏幕和铰链连接件。底座是整机的水平承托面，直接用一个居中的长方体表达，不需要内部装配。

屏幕是绕底部铰链方向打开的薄面板。它的厚度和宽度保持在面板自身局部坐标中，开合只改变面板相对底座的倾角。装配时保留一个竖直方向的铰链投影基准，让屏幕底边在高度方向贴到这个基准；同时保留屏幕自身的中心深度平面。这样可以清楚地区分“面板开合角”和“面板在笔记本深度方向上的放置平面”。

连接件是沿笔记本宽度方向排列的铰链段。矩形连接件从每段的左端中心开始逐段铺开，圆柱连接件则把圆柱轴线直接对齐到每段的两个端点。相邻段之间的间距作为阵列规则，而不是在每个部件里重复计算累计和。
