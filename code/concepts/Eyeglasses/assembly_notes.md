# Eyeglasses 装配思路

Eyeglasses 的重写以“左右成对的局部部件”为核心：镜片、镜框、镜腿和鼻托都先在自己的局部坐标中定义形状，再通过左右符号生成一对对称部件。整体 category 级姿态只在最后统一应用。

梯形镜片用一个带顶边偏移的薄 Cuboid 表达。左右镜片共享同一轮廓，只改变顶边偏移和局部姿态符号；鼻梁间距由镜片内侧的局部参考点控制，而不是把参考点作为孤立顶点塞进 mesh。

梯形镜框先在单侧局部 frame 中分解为上框、下框、内侧框和外侧框。框边宽度沿梯形轮廓投影得到各条框边的实际尺寸；单侧 frame 完成后，再作为整体绕镜片姿态旋转并生成左右两侧。

圆形镜片和圆形镜框都把圆柱或圆环的高度轴转为镜片厚度方向。左右间距由椭圆外轮廓的内侧边缘决定，避免依赖采样顶点编号。

镜腿是局部旋转链：主段决定整条镜腿相对镜框的姿态，末端段先在主段局部 frame 中折转，再跟随主段一起放置到左右铰链位置。三折镜腿在此基础上增加一个独立连接块；连接块用于铰链安装，不参与末端折转。

鼻托支撑件是简单的镜像 pair。连接块以前表面作为安装面：单连接块贴到中心安装面，双连接块分别贴到各自给定的安装点。
