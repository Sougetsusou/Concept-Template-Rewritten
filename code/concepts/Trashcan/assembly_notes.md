# Trashcan 装配说明

Trashcan 的重写把几何分成三类：桶身、盖子和轮子。桶身负责形成容器空间，盖子负责封闭或开孔，轮子只表达一组关于中心镜像的轴向圆柱。

圆柱桶身由底部实心过渡段和上部中空环组成。过渡处半径由外轮廓在对应高度上的线性插值得到；底部圆柱的顶面和上部环的中心轴底面通过 anchor 相接。带分隔板的圆柱桶身在同一个过渡高度上竖起一块矩形隔板，因此隔板底面也贴到实心段顶面。

棱柱桶身由一个矩形环壳和内部底板组成。环壳表达外轮廓、内腔、顶面偏移和前后高度差；底板的顶面尺寸按壳体高度比例从内轮廓插值得到，并用底面 anchor 对齐到壳体底部平面。

普通盖子是单个圆柱、单个斜棱柱、单个中空环或两层 cuboid 的组合。双层盖以下层底面作为基准，上层底面贴到下层顶面并带水平偏移。

带孔圆柱盖由底部若干弧形环段、中部完整环和顶部实心圆盘组成。弧形环段按等角径向阵列分布；中部环和顶部圆盘沿中心轴逐层堆叠。

带孔矩形盖由顶板和四个侧面组成。每个侧面根据开孔开关选择整板，或拆成两根侧柱和一根上梁。前后侧围绕 Z 方向镜像，左右侧围绕 X 方向镜像。左右侧开孔上梁在 rewritten 中使用侧壁自身厚度作为 X 向跨度，这是实际侧壁装配语义。

轮子是一对沿 X 方向镜像的圆柱，圆柱主轴旋转到 X 方向，轮距参数表示每个轮心到中心平面的距离。
