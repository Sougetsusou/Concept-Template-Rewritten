# Lighter 装配说明

Lighter 的 rewritten 版本把几何拆成几个稳定的部件族：机身、砂轮、按钮、喷口罩、盖体和灯芯。每个模板只描述自身局部结构，整体 `position` 和 `rotation` 作为 concept 级姿态统一应用。

机身有三种轮廓。纯矩形机身是一个居中长方体；圆弧机身是在矩形主体的前后两侧贴半圆柱，让主体平面和弧面平直接触；双层机身是在主体顶面上方叠加一个可在平面内偏移的顶层。

砂轮沿横向轴排布。简化砂轮只是一段横向圆柱；标准砂轮以中间轮为主体，两端各贴一个较窄侧轮，三段保持同轴。

按钮由上下两层堆叠形成。普通 L 形按钮让上层相对下层在深度方向偏移；带拱面的按钮在上下两层背侧各贴一段半圆柱，保留两层按钮自己的接触平面。

喷口罩有三种形式。矩形喷口由左右侧壁、前壁和顶部盖片围成；拱形喷口把前壁替换成半圆柱拱面；围合喷口直接使用矩形环壳表达中空外罩。

盖体由下部矩形环壳和顶部盖片组成，整体开口相对外壳向一侧偏置。灯芯是竖直圆柱，底面作为安装基准。
