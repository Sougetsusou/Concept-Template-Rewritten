# Scissors 装配说明

剪刀被拆成轴心、刀刃和手柄三组模板。轴心模板描述两片剪刀在铆接处的上下分层关系：长方体轴片按左右层相反方向偏转，必要时在中心放入圆柱铆轴；圆柱轴心则直接表达上下两层圆柱的叠合。

刀刃模板以根部楔形段为基准，沿剪刀前方延伸。尖端可以是继续收尖的直线段，也可以是四分之一弧形段。根部的横向偏移属于刀刃截面形状的一部分，而不是额外装配关系。

手柄模板统一表达为“根部连接块 + 握持轮廓”。根部连接块先确定握环的局部安装原点，后续完整环、半环、多段弧形环、多边形环或直柄都围绕这个原点构建。弧形手柄中的半径、存在角、中心距和条带厚度共同决定握环轮廓，因此保留为 class-local shape derivation。

特殊标记：`_RootedScissorsHandleFrame` 是剪刀手柄专属局部 frame，用来表达“根部连接块之后进入握持轮廓”的安装坐标系。它不是公共 relation，也不应直接迁移到其他 category；只有出现同类根部连接件和握持轮廓语义时，才重新评估是否抽象为更通用的 frame 能力。
