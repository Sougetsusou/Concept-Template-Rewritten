# Refrigerator rewritten 装配思路

Refrigerator 的 rewritten 装配把冰箱看成一组矩形盒体、门板、把手、托盘和支脚的组合。主体类先确定背板或底座这样的实体基准件，再把开口矩形框通过真实面 anchor 接到基准件上；内部分隔板挂到背板的内侧平面，并保留上下分层或左右分层的不同方向语义。

门板类以背面安装平面为局部基准。普通门板和普通把手都是从该平面向前伸出；凹陷门板则由后部实体层和前部矩形框组成，前框直接接到实体层的前表面，凹陷开口通过矩形环的内孔偏移表达。

直把手由两个安装脚和前方握持条组成。安装脚成对布置在竖向两侧，握持条从安装脚前表面继续向前延伸。弧形把手使用 `Ring` 表达圆弧段。三段弧形把手的安装脚间距用于确定弧心深度，弧段长度仍由显式存在角控制；这表示一个长弧把手和两个安装脚的组合，而不是把弧段端点强行绑定到安装脚。单段弧形把手同样保留显式弧角。弦高和半角姿态属于把手自身的形状定义，保留在 class 内部，而不是在外部手算旋转后的顶点。

容器和抽屉托盘都按“下部实体底座 + 上部开口矩形环”装配。容器版本把前缘平面对齐到局部基准面，抽屉托盘版本保持整体以自身中心深度对称。平托盘直接是一块矩形板。

支脚类使用腿顶面作为安装 anchor。不同数量的腿只改变安装点阵列：单腿居中，两腿左右成对，三腿是前排两腿加后方中心腿，四腿是前后两排左右腿。每条腿都从安装平面向下延伸。
