# USB 装配思路

USB 主体以一个前后方向明确的矩形体作为基准。矩形体前面作为开口/连接方向，后面作为圆角和端帽的安装面；后端圆角、左右侧圆角和后角补片都贴在这个基准体的真实面或边中心上。

连接头分成实心和中空两类。实心连接头是单一块体，从安装面向外伸出；中空连接头用一个矩形环表达外框和偏置内孔，环的深度轴旋转到 USB 的前后方向后再整体放到安装面前方。

普通帽体由前侧中空套筒和后侧实心封闭块组成。套筒负责提供插入口，封闭块直接接在套筒末端，整体沿前后方向形成一段开口、一段封闭的壳体。

旋转帽体由上下两条倾斜直臂和后侧弧形桥组成。弧形桥定义两条直臂的后部连接关系；方端版本让直臂直接结束，圆端版本在直臂远端增加半圆端帽。可选 shaft 是一对上下镜像的圆柱轴，安装在同一深度位置。

旋转帽中的半径、倾角和局部平面朝向是该模板自己的形状定义。弧形桥使用复合旋转矩阵表达“先成为后侧弧，再放进帽体平面”的局部 frame，不再在 concept 里直接改写 vertices。
