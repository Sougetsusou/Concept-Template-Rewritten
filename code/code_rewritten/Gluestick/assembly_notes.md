# Gluestick 装配思路

Gluestick 的主体模板都可以看成一段连续外壳：圆柱主体用椭圆截面的圆柱或锥台表达，压扁主体用一端为圆、一端收束成扁平线段的渐缩体表达，方形主体直接用长方体表达。这几类主体没有内部拼接关系，重点是保持局部主轴和最终姿态约定一致。

半球盖由两段组成：下方是一段筒状底座，上方是一段被纬线截断的球冠。装配时先把筒状底座的底面放在局部原点，再把球冠的截断圆心贴到底座顶面。这个关系表达的是“球冠坐在筒口上”，不是手动复刻旧代码里的中心偏移公式。

圆筒盖由下部开口环壁和上部封闭盖顶组成。环壁先以底面作为装配基准，盖顶再用底面贴到环壁的顶面；如果外壁是锥台，环壁顶部的外半径由整体外轮廓在开口高度处插值得到。这是盖子自身的形状推导，保留在模板内部。

所有模板的最终姿态使用同一套 gluestick 局部坐标约定：局部 Y 轴是高度方向，category 级变换继续采用原数据的先平移后旋转姿态语义。
