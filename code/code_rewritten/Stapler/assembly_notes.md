# Stapler 装配思路

## SideBlockStaplerBody

主体由一个长底座和两个上方侧块构成。底座提供支撑面，侧块贴在底座上表面并靠近后侧边界，左右侧块通过相同间距形成镜像。可选横轴被看作沿左右方向串接的三段同轴圆柱：中段居中，两端段分别贴到中段两端。

## SolidStaplerCover

简化盖板是一个单体长方体。它的局部基准放在后下边线上，使盖板自然从基准向上、向前延伸。

## CarvedStaplerCover

开槽盖板由下方矩形开口框和上方实心盖段组成。下框定义开槽高度和内外轮廓，上盖段直接叠在下框顶面，形成完整外包络。

## CarvedStaplerMagazine

开槽钉匣使用与开槽盖板相同的上下堆叠思路，但语义上作为钉匣槽体的一种简化版本。外框决定槽体外轮廓，内部开口决定可容纳钉排的空间。

## BoxTrayStaplerMagazine

复杂钉匣被视为一个开放槽体：底板承托钉排，后板和前挡板限定深度方向边界，左右侧板限定宽度方向边界。侧板允许沿深度方向单独偏移或缩短，因此它不是一个完整封闭盒，而是由五块板按接触面装配成的托槽。
