# Safe 装配思路

保险柜外壳用“后侧实心板 + 前侧中空框”表达。后板先锚到整体背面，前框再贴到后板前表面；内孔偏心由上下、左右厚度差自然决定。

多层柜体在同一外壳内增加隔板。隔板都以后表面贴到内腔背面，横向和竖向位置由内腔中心与各自偏移决定。

门板类统一把局部 Z=0 当成安装基准。单层门以后表面贴到基准面；后层门和前层门分别在基准面的负 Z 或正 Z 侧增加附加板；凹陷门由后侧实体板和前侧偏心矩形框组成。

连接件保持简单的主体-附件关系。圆柱连接件是单个圆柱；T 形连接件以圆柱为中心件，侧向块用边中心锚贴到圆柱外侧的切向安装点。

把手按形态分为三类。三折把手由两个平行底条和一个前桥组成；爪形把手在圆柱底座前方做等角径向叉爪阵列；圆形把手在叉爪外端再加一圈圆环，圆环内径跟随叉爪外端径向投影。

旋钮由沿 Z 方向伸出的底部锥台和顶部圆柱端面对齐堆叠。控制器是单个以后表面贴到安装面的面板。

支腿模板保留“腿数量决定拓扑”的语义：一腿居中、二腿左右镜像、三腿为前双后单、四腿为前后两组镜像。每条腿都用顶面中心锚接到安装平面，然后向负 Y 方向延伸。
