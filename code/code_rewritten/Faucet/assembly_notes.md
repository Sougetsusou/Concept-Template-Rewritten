# Faucet Assembly Notes

## 总体思路

Faucet 被组织成三类部件：基座、喷口和开关。基座负责提供安装面或支撑形体；喷口负责表达水流路径；开关负责表达可操作的旋钮、手柄或杆件。所有 class 都保留自己的最终位姿，局部装配完成后再统一应用整体平移和 `YXZ` 旋转。

## 基座

方形和圆柱基座是简单的可选双层堆叠：第一层定义主安装体，第二层在需要时沿局部负 Y 方向继续接出，并允许一个相对偏移。圆形基座是单个沿深度方向放置的圆柱。U 形基座由左右支撑和顶部横梁组成，左右部件以中心平面对称，横梁连接两侧端部，形成 XZ 或 YZ 平面中的 U 形轮廓。弧形基座使用弦长和弧高定义一段圆弧管，再把这段管放到基座局部姿态中。

## 喷口

折线喷口被看作一条由多个控制点组成的管线。每段圆管直接把自身局部高度轴对齐到相邻控制点，不再在 concept 层手写端点旋转角。花洒喷口在折线路径末端可选添加一个喷头。弧形喷口由底部直管、圆弧管和末端直管组成：底部直管连接起点和弯曲起点，圆弧管表达转弯路径，末端直管沿转弯后的方向延伸。

## 开关

简化开关、旋钮和盒状手柄主要是按 Y 或 Z 方向堆叠多个基础体。常规开关由一个主圆柱和一个切向副圆柱组成，副圆柱依据主圆柱半径和自身半径贴在外侧。T 形开关由两段竖直圆柱和顶部横杆组成，顶部横杆可绕 Y 轴旋转。杠杆开关由底座和两段折线圆管组成。

旋转式开关由一到两个主轴体和一组径向分布的拨片组成。拨片先定义自身局部倾斜，再按指定轴向做等角阵列；阵列只负责角度和方向，安装中心仍由当前开关的局部安装点给出。

## 验证说明

折线管和弧形喷口的 legacy 源码会在向量分量为零时加入 `1e-5` 量级的防零补偿。rewritten 版本直接连接真实控制点，因此这些 class 的可视化比较允许 `2e-5` bbox 容差；其余 class 继续使用严格 bbox 对齐。
