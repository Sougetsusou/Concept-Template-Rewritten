# KitchenPot 装配说明

KitchenPot 的重写版本把锅身、盖子、顶把手和侧把手都放在各自的局部坐标系内装配，最后再由 concept 级位姿放到整锅坐标中。整体规则是：轴向壳体从底部或开口基准面向上堆叠，把手先在自己的安装平面上成形，再整体定位到锅体上。

锅身由下部封闭段和上部开口环组成。下部段先确定总高度的底面，上部环的中心轴底点贴到下部段顶面，从而保持外轮廓连续。带内挖的圆柱盖使用相同的轴向思路，只是先放下方开口环，再把上方封闭盖贴到开口环顶部。

球冠盖把截断球面的底部纬线作为安装基准，移动到局部基准平面上。这样盖子的形状仍由球面截断角决定，而不是在外层手写一个额外的中心补偿。

顶把手分为单段、三折、半环和多层旋钮。单段把手从安装平面向上伸出；三折把手用左右两个倾斜支脚确定支撑高度，再把顶部握持横梁放在支脚顶端高度；半环把手保持为一个有方向的弧形环；多层旋钮按每层底面到上一层顶面的顺序向上堆叠。

侧把手分为三折、L 形和单段。三折侧把手先在局部平面里构造前后两根安装杆和一根握持桥，再把这一组三件绕自己的 pivot 旋转并镜像到另一侧。L 形侧把手用左右两组下臂和上臂表达，边缘 anchor 保证上臂接在下臂顶边。单段侧把手则让前表面贴到安装平面，实体向后延伸。
