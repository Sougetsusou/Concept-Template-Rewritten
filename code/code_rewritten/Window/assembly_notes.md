# Window 装配思路

Window 的主体由平面窗框和玻璃片组成。外窗框是一个矩形中空环；每个子窗扇也是一个矩形中空环，玻璃片放在对应窗扇开口的中心偏移位置。窗扇类只负责决定每个窗扇属于哪一层、在哪个水平或深度位置，以及玻璃偏移是否需要镜像。

对称窗按数量选择固定的窗扇布局：一到六扇窗共享三种窗扇模板，并按中层、上层、下层组合成左右结构。非对称窗按竖向层级顺序堆叠多个窗扇。竖向滑动窗则把两扇窗放在相邻高度层，并允许它们在深度方向错开。

把手都挂在某个窗扇层位的安装平面上。矩形把手用一个方体贴到安装平面；拱形把手用两个脚块和一段圆弧桥连接；L 形把手从安装平面开始连续接出底段、中段和顶段。不同把手保留各自的层位解释，因为它们在 legacy 设计里并不完全共享同一套前后安装规则。
