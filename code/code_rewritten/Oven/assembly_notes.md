# Oven 装配说明

Oven 的 rewritten 版本按功能分成箱体、门、把手、托盘、挡板、控制面板、炉顶和支脚。每个模板只描述自己的局部装配，最终位置和姿态作为 concept 级变换统一应用。

箱体由后侧实体板和前侧开口壳体组成。后板先确定完整深度的背面基准，开口壳体再贴到后板前侧，矩形内孔表达烤箱正面的开放空间。双层箱体在同一个开口壳体内增加一块分隔板，分隔板从后侧基准向前伸入腔体。

门板有平板和下沉框两种。平板门从背面安装面向前延伸；下沉门先放置后侧实体层，再把前侧矩形框贴到实体层前面，框内开口可以在门板平面内偏移。

把手都以背面安装面为基准。直把手是单个长方体；三段式把手用上下安装座和前方握持条组成；弧形把手用一段圆弧表达握持段，弧段的半角、弧心深度和安装座间距共同决定它与安装座的相对位置。

托盘分为平板和抽屉式。平板托盘保持中心放置；抽屉式托盘先放底板，再把上方矩形开口壁贴到底板顶部。

控制面板由倾斜底板和按钮列表组成。按钮的偏移沿面板斜面解释，每个按钮用自身背面贴到面板前表面，并按面板斜率旋转。

炉顶先放置平板，再把每个炉头的外环和中心圆柱贴到平板顶面。支脚以顶部安装点为基准向下延伸，并根据腿数量选择中心、左右成对、前三点或前后四点布局。
