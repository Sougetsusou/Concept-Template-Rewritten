# Pen 装配说明

Pen 的主体结构以笔的中心轴为基准。笔杆和笔帽都按轴向壳体处理：单段壳体直接使用中空圆环，双段壳体把下一段的顶部中心轴贴到上一段的底部中心轴。笔帽下部是封闭实体段，上部是中空壳体，两者在同一个截面半径处相接。

直线笔夹是一个局部平面内的折线结构。根部先贴到安装基准，外侧长段从根部外缘继续向下延伸，末端短段再回折到靠近根部的一侧。它不是旋转关系，而是三段矩形片在同一个局部平面中的端边连接。

弧形笔夹由一段圆环弧表达。覆盖角决定弧段本身的形状，也决定弧心相对安装坐标的姿态，因此保留为笔夹模板内部的形状推导，而不是放入参数 mapper。

按钮和笔芯都按轴向堆叠处理。按钮以底面作为安装面向上增长；双层按钮把上层底面贴到下层顶面。笔芯的主圆柱保持居中，下端依次接细连接段和倒置锥尖，锥尖的底面贴到连接段末端。锥尖偏移属于锥体自身的局部底面语义；笔尖倒置后，偏移方向随锥体姿态一起翻转，并由非零偏移测试锁定。
