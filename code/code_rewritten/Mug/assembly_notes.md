# Mug 装配说明

杯身模板都以杯子的中心竖轴为基准。封底段先固定到整体底面，上方开口壁再用轴心底部锚点贴到下段顶部；多层杯身继续沿同一轴心逐段叠加，每一层的底部轮廓来自上一层顶部轮廓，从而保持内外壁连续。

方形杯身保留“用半径式尺寸控制方形轮廓”的设计：轮廓尺寸先转换成方形边长，再按与圆形杯身相同的封底段加开口壁结构装配。

三段把手把上下横段看成两个从杯身安装点伸出的短梁。前方连接段读取两个横段的前端锚点，用这条安装线确定自身方向，并把自身背面中线贴到安装线上；连接段长度保留横段厚度带来的搭接量。

弯曲把手由单个弧形管件形成，类别层只负责设置弧段方向和整体姿态，不把弧形 primitive 拆成额外关系。

单圆柱构件没有内部装配关系，只保留圆柱半径、高度和整体姿态。
